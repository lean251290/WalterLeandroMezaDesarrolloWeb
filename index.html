DISEÑADOR WEB

Generalidades del curso.

fundamentacion:
objetivos
Al finalizar este curso se espera que los estudiantes puedan:
  • Crear paginas utilizando HTML, CSS, SVG y JavaScript
  • Escribir el 100% de un sitio web
  • Diseñar una web que se addapte a diferentes tamaños de pantalla y gestionar la navegacion
  • Crear elementos animados que interactúen con el usuario
  • Consumit API's con JavaScript para colocar contenidos en un sitio

Contenido

UNIDAD 1: HTML Y DESARROLLO WEB

Entorno de desarrollo. Editor de código. Inspector y herramientas de desarrolladro en los navegadores. Funcionamiento básico de un sitio web. Elementos de un sitio web. Etiqeutas y atributos. Etiquetas de texto. Imágenes y enlaces. Etiquetas de bloque. Etiquetas de documento. Repositorios y Git.

UNIDAD 2: INTRODUCCION A CSS

Vincular un archivo CSS. Sintáxis básica de css. Selectores. Tipografía y estilos. Estructura en las hojas de estilo. Selectores avanzados. Estilos de navegación. Rests CSS. El modelo de la caja. Display. Floats. Position. Estilos para las listas. Border, border-radius y box-shadow. Fondos e imagágenes de fondo. WebFonts. Colores y opacidad. Bordes y redondeados. Sombras. Degradados y fondos.
